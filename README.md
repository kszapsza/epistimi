# epistimi

A platform combining electronic school diary and e-learning features.

## Tech stack

**Backend:**

* 🍅 Kotlin
* 🍃 Spring Boot
* 🧪 Kotest
* 📦 MongoDB

**Frontend:**

* ⚛️ TypeScript & React
* 📦 Redux
* 🧪 Jest & React Testing Library
* 🎨 SCSS, Mantine UI
